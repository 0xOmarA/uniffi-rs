FfiConverter for Arc-types

This trait gets around the orphan rule limitations, which prevent library crates from implementing FfiConverter on an Arc. When this is implemented for T, we generate an FfiConverter impl for Arc.

Note: There’s no need for FfiConverterBox, since Box is a fundamental type.

Safety

This has the same safety considerations as FfiConverter
